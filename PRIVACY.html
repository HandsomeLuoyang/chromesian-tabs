Privacy Policy

English

Bookmark Home does not collect, store, or transmit any personal data.

  • All data is stored locally via Chrome's storage.local API.
  • No information is sent to any server, API, or third party.
  • The bookmarks permission is used solely to read and display your Chrome bookmarks on the new tab page — data never leaves your browser.
  • The favicon permission displays website icons via Chrome's built-in _favicon/ endpoint.
  • No analytics, tracking, or telemetry is included.

You can clear all stored usage data anytime via the settings panel. Removing the extension deletes all associated local data.

隐私政策

中文

Bookmark Home（书签主页） 不会收集、存储或传输任何个人数据。

  • 所有数据均通过 Chrome 的 storage.local API 本地存储在您的设备上。
  • 没有任何信息发送到任何服务器、API 或第三方。
  • bookmarks 权限的唯一目的是在新标签页上展示您的 Chrome 书签——数据不会离开您的浏览器。
  • favicon 权限用于通过 Chrome 内置端点显示网站图标。
  • 不含任何形式的分析、追踪或遥测。

您可以随时通过设置面板清空所有使用数据。移除扩展程序将删除所有本地数据。
