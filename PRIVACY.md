# Privacy Policy / 隐私政策

**Last updated: 2026-06-24**

## English

**Bookmark Home** does not collect, store, or transmit any personal data.

- All data (bookmarks, usage statistics, aliases, settings) is stored **locally** on your device via Chrome's `storage.local` API.
- No information is sent to any server, API, or third party.
- The extension requires the `bookmarks` permission solely to read and display your Chrome bookmarks on the new tab page — this data never leaves your browser.
- The `favicon` permission is used to display website icons (favicons) for your bookmarks via Chrome's built-in `_favicon/` endpoint, which requires no external network requests beyond what the browser already makes.
- No analytics, tracking, or telemetry of any kind is included.

### Data you can control

You can clear all stored usage data at any time via the settings panel (⚙ → "使用统计" → "清空"). Removing the extension will delete all associated local data.

## 中文

**Bookmark Home（书签主页）** 不会收集、存储或传输任何个人数据。

- 所有数据（书签、使用统计、别名、设置）均通过 Chrome 的 `storage.local` API **本地存储**在您的设备上。
- 没有任何信息发送到任何服务器、API 或第三方。
- 扩展程序需要 `bookmarks` 权限的唯一目的是在新标签页上读取和展示您的 Chrome 书签——这些数据永远不会离开您的浏览器。
- `favicon` 权限用于通过 Chrome 内置的 `_favicon/` 端点显示网站图标，不需要超出浏览器本身已发出的外部网络请求。
- 不含任何形式的分析、追踪或遥测。

### 您可以控制的数据

您可以随时通过设置面板（⚙ → "使用统计" → "清空"）清除所有已存储的使用数据。移除扩展程序将删除所有相关的本地数据。
